How to operate?

  1. You can count numbers by using arithmetic signs (+, –, ×,÷ etc.).
  2. If you want to use the root (√) sign at first you have to give the numbers' you want to count. Then, press the root (√) button.
  3. Press AC button to clear all
  4. Press DEL button to delete numbers or sings by one.
  5. If you want to see the reminder of any devide you have to use the Modular (%) button.
  6. To count power you have to press the Exponentiation (x x) button. Then, you have to give the value of power. Then, press the Equal (=) button.
